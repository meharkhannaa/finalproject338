Accessibility Audit

Accessibility Manual & Automated Testing

    • Role: Automated Testing Lead
      Timeline: October 2025
      Tools: Wave, aXe

    • Problem Statement:Culturally Situated Design Tools (CSDT) wants to make sure to address accessibility concerns to make sure that their website is accessible to all varying abilities and identities.

    Process Outline:We began with automated testing, then transitioned to manual testing. We performed a WCAG 3.1 Accessibility Audit and delivered our findings and recommendations for improvement in a presentation and subsequent report to the website's organization lead.

    Testing and Debugging:We performed both automated and manual testing for each page of the website.

    Challenges:The website was very image heavy and no images had alternative text. To address this, we created a how-to guide on alt text captions and provided 45+ examples with real images from the website.
